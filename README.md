# 🛡️ PixVault

## AI-Powered Deepfake Prevention & Privacy Protection System

PixVault is an AI-powered image protection platform designed to safeguard personal photographs from unauthorized AI training, facial recognition systems, identity cloning, and deepfake generation.

The system leverages adversarial machine learning techniques to subtly modify images in ways that remain visually similar to humans while significantly reducing the effectiveness of AI models attempting to learn facial representations.

---

# 🎯 Problem Statement

With the rapid advancement of Artificial Intelligence and Deepfake technologies, publicly shared images can be collected and used without the owner's consent.

These images can be exploited to:

* Train facial recognition systems
* Generate realistic deepfakes
* Clone personal identities
* Create synthetic media
* Impersonate individuals online
* Compromise digital privacy

Currently, users have limited control over how their images are used after publication.

PixVault addresses this challenge by protecting images before they are shared online.

# 🔬 Protection Techniques Implemented

## Fawkes Protection

Generates adversarial perturbations that prevent facial recognition systems from learning accurate facial representations.

## LowKey Protection

Applies imperceptible image modifications that disrupt AI feature extraction while maintaining visual quality.

## Ulixes Protection

Protects identity by manipulating feature-space representations used by machine learning models.

## AMT-GAN Inspired Protection

Uses adversarial transformations to increase resistance against deepfake training pipelines.

## Combined Protection Engine

Combines multiple protection strategies into a single defense pipeline for enhanced robustness.

# ⚙️ Installation

## Clone Repository

```bash

```

## Create Virtual Environment


### Windows


```bash
source venv/bin/activate
```

```bash
pip install -r requirements.txt
```

```bash
python app.py
```

or

